Git é um sistema de controle de versão distribuído, amplamente usado para gerenciar o histórico de mudanças em arquivos de projetos. 
O Git funciona criando snapshots (fotos) dos arquivos em um dado momento, armazenando apenas as mudanças feitas entre essas versões. 
Cada colaborador pode ter uma cópia completa do projeto, facilitando o trabalho offline e permitindo o desenvolvimento em paralelo por meio de branches (ramificações).
As vantagens de usar o Git são: controle e segurança, colaboração e organização.
